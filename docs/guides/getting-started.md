# 快速开始

## 首次启动

1. 启动 OpenClaw 应用
2. 应用会显示启动屏幕
3. 等待后端服务启动（通常 5-10 秒）
4. 应用加载完成后，你会看到登录界面

## 登录

1. 输入用户名或邮箱
2. 输入密码
3. 点击"登录"按钮
4. 如果是首次登录，可能需要进行身份验证

## 基本配置

### 设置用户信息

1. 点击右上角的用户头像
2. 选择"设置"
3. 更新你的个人信息
4. 点击"保存"

### 配置工作区

1. 点击左侧菜单的"工作区"
2. 点击"新建工作区"
3. 输入工作区名称
4. 选择工作区类型
5. 点击"创建"

## 常见问题

### 应用无法启动？
- 检查系统要求是否满足
- 查看 [故障排查指南](../../TROUBLESHOOTING.md)

### 无法连接到后端？
- 检查网络连接
- 查看防火墙设置
- 重启应用

### 忘记密码？
- 点击登录界面的"忘记密码"
- 按照邮件指示重置密码
