# 基本操作

## 导航

### 主菜单

左侧菜单包含以下主要功能：

- **仪表板** - 查看概览和统计信息
- **项目** - 管理项目和任务
- **团队** - 管理团队成员
- **文档** - 创建和编辑文档
- **设置** - 应用设置

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` / `Cmd+K` | 打开命令面板 |
| `Ctrl+/` / `Cmd+/` | 显示快捷键帮助 |
| `Ctrl+S` / `Cmd+S` | 保存 |
| `Ctrl+Z` / `Cmd+Z` | 撤销 |
| `Ctrl+Y` / `Cmd+Y` | 重做 |

## 常见操作

### 创建项目

1. 点击"项目"菜单
2. 点击"新建项目"
3. 输入项目名称和描述
4. 选择项目模板（可选）
5. 点击"创建"

### 创建任务

1. 进入项目
2. 点击"新建任务"
3. 输入任务标题
4. 添加描述和附件（可选）
5. 分配给团队成员
6. 设置优先级和截止日期
7. 点击"创建"

### 邀请团队成员

1. 点击"团队"菜单
2. 点击"邀请成员"
3. 输入成员邮箱
4. 选择角色
5. 点击"发送邀请"

## 搜索和过滤

### 全局搜索

按 `Ctrl+K` / `Cmd+K` 打开搜索框，输入关键词搜索项目、任务、文档等。

### 过滤

在列表视图中，点击"过滤"按钮，选择过滤条件：

- 状态
- 优先级
- 分配人
- 截止日期
- 标签

## 导出和分享

### 导出数据

1. 选择要导出的项目或任务
2. 点��"更多"菜单
3. 选择"导出"
4. 选择导出格式（CSV、PDF、JSON）
5. 点击"导出"

### 分享

1. 选择要分享的项目或文档
2. 点击"分享"按钮
3. 选择分享方式（链接、邮件、团队）
4. 配置权限
5. 点击"分享"
